The Button component triggers an action or event, such as submitting a form, opening a dialog, or canceling an action. It supports polymorphism, allowing it to render as a `<button>`, `<a>`, or `<span>`.

## Variants

We support four visual hierarchies.

<Playground>
  <div style="display: flex; gap: 1rem; flex-wrap: wrap; align-items: center;">
    <Button variant="primary">Primary</Button>
    <Button variant="secondary">Secondary</Button>
    <Button variant="ghost">Ghost</Button>
    <Button variant="text">Text Link</Button>
  </div>
</Playground>

```tsx
<Button variant="primary">Primary</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="ghost">Ghost</Button>
<Button variant="text">Text Link</Button>
```

## Sizes

Buttons are available in three fixed sizes.

<Playground>
  <div style="display: flex; gap: 1rem; align-items: center;">
    <Button size="sm">Small</Button>
    <Button size="md">Medium</Button>
    <Button size="lg">Large</Button>
  </div>
</Playground>

```tsx
<Button size="sm">Small</Button>
<Button size="md">Medium</Button>
<Button size="lg">Large</Button>
```

## Icons

Buttons support a single slot for composition. You can place icons before or after the text, and flexbox handles the spacing automatically.

<Playground>
<div style="display: flex; gap: 1rem;">
<Button variant="secondary">
  <Icon  label="Go back">
    <ArrowLeft />
  </Icon>
  Go Back
</Button>

<Button variant="primary">
  Next Step
  <Icon  label="Next step">
    <ArrowRight />
  </Icon>
</Button>
</div>
</Playground>

### Icon Only

Use the `iconOnly` prop to enforce a square aspect ratio.

It can be used in combination with the different variants and sizes

<Playground>
  <div>
    <Button variant="secondary" iconOnly aria-label="Menu">
      <Icon>
        <Menu />
      </Icon>
    </Button>
  </div>
  <div>
    <Button variant="ghost" size="lg" iconOnly aria-label="Menu">
      <Icon>
        <Menu />
      </Icon>
    </Button>
  </div>
</Playground>

```tsx
<Button iconOnly aria-label="Menu">
  <Icon>
    <Menu />
  </Icon>
</Button>

<Button variant="ghost" size="lg" iconOnly aria-label="Menu">
  <Icon>
    <Menu />
  </Icon>
</Button>
```

## Layout & Responsive

### Full Width

Forces the button to take up 100% of its container.

<Playground>
  <Button fullWidth>Sign In</Button>
</Playground>

### Mobile Only Full Width

The button behaves normally on desktop but expands to full width on screens smaller than the `sm` breakpoint.

<Playground>
  <Button fullWidth="mobile-only">Full Width on Mobile</Button>
</Playground>

```tsx
<Button fullWidth="mobile-only">Checkout</Button>
```

## Polymorphism

The component automatically renders as an `<a>` tag if you provide an `href`. You can also force specific tags using the `as` prop.

<Playground>
  <div style="display: flex; gap: 1rem;">
    <Button href="#" target="_blank">
      I am an Anchor
    </Button>
    <Button as="span" variant="ghost">
      I am a Span
    </Button>
  </div>
</Playground>

## API Reference

| Prop        | Type                                            | Default     | Description                          |
| :---------- | :---------------------------------------------- | :---------- | :----------------------------------- |
| `variant`   | `'primary' \| 'secondary' \| 'ghost' \| 'text'` | `'primary'` | The visual style of the button.      |
| `size`      | `'sm' \| 'md' \| 'lg'`                          | `'md'`      | Controls padding and font-size.      |
| `href`      | `string`                                        | `-`         | If present, renders as an `<a>` tag. |
| `fullWidth` | `boolean \| 'mobile-only'`                      | `false`     | Controls width behavior.             |
| `iconOnly`  | `boolean`                                       | `false`     | Enforces square aspect ratio.        |
| `as`        | `'button' \| 'a' \| 'span'`                     | `-`         | Overrides the HTML tag.              |
